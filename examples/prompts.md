# Prompt Examples

These prompts can be copied directly into Codex.

## Illustration Planning Only

```text
Use $ian-xiaohei-illustrations — don't generate images yet.
Please analyze where this article would benefit from illustrations, and output a shot list of about 5 images.
For each image, specify:
- Which paragraph it follows
- The image theme
- Core meaning
- Structure type
- What Xiaohei is doing in the image
- Suggested elements
- Suggested Chinese annotation words

<paste article>
```

## Article Body Illustrations

```text
Use $ian-xiaohei-illustrations to generate 4 quirky Xiaohei body illustrations from the article below.
Requirements: landscape 16:9, pure white background, black hand-drawn line art, small amounts of red/orange/blue handwritten Chinese annotations.
Each image should convey only one core structure — no PPT infographics, no cute cartoons.

<paste article>
```

## Long-Form Illustration Strategy

```text
Use $ian-xiaohei-illustrations to create an illustration strategy for this long article.
Don't distribute images evenly — only pick cognitive anchors: core judgments, input/output loops, before/after comparisons, common pitfalls, handoff paths.
Default 6-8 images. Output the shot list first — don't generate images yet.

<paste article>
```

## Single Concept, One Image

```text
Use $ian-xiaohei-illustrations to generate a 16:9 body illustration for this idea:

"Trust isn't shouted into existence — it's paved with evidence, piece by piece."

The image should be quirky but clean. Xiaohei must perform the core action.
Chinese annotations: maximum 5, keep them short.
```

## Workflow Theme

```text
Use $ian-xiaohei-illustrations to generate an image for: "Processing a single raw piece of material into three types of content: traffic, trust, and conversion."
Don't draw a formal flowchart. Don't replicate the old 'one fish, many uses' example.
Please reinvent a new low-tech metaphor with Xiaohei participating in the core action.
```

## Edit Image: Remove Title

```text
Use $ian-xiaohei-illustrations to edit this image.
Remove the "Workflow / 流程图" title and underline from the top-left corner. Keep everything else unchanged.
Don't add any new text or objects.
```

## Edit Image: Strengthen Xiaohei's Participation

```text
Use $ian-xiaohei-illustrations — this image is on the right track, but Xiaohei feels like decoration.
Keep the core meaning unchanged, but regenerate a version where Xiaohei becomes the one truly driving the structure's operation.
Make it stranger, but still pure white, clean, and minimal text.
```

## Generate a Style Sample Set

```text
Use $ian-xiaohei-illustrations to produce 5 different themed Xiaohei body illustration samples.
Themes: information overload, product validation, content compounding, one-person company, trust building.
Generate each separately — don't compose into a single image.
```
