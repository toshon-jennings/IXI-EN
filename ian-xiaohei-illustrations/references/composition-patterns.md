# Composition Patterns and Originality Rules

## Basic Structure Types

Choose one structure per image — don't mix too many.

### Workflow

Best for: Input → Processing → Output, content production, AI workflows, automated pipelines.

How to draw: input on the left, Xiaohei or a strange machine processing in the middle, output on the right, orange arrows showing main flow direction.

### System Detail

Best for: Information sources, filters, databases, renderers, local views of agent systems.

How to draw: only 3-5 core modules, Xiaohei participating in one key action.

### Before / After Comparison

Best for: chaos/order, manual/automatic, scattered/focused, anxiety/stability.

How to draw: chaos on the left, stability on the right, orange arrow in the middle. Characters can be more exaggerated.

### Character State

Best for: User pain points, creator states, tool overload, information anxiety, going from stuck to running.

How to draw: 2-4 small states, each with a short annotation.

### Concept Metaphor

Best for: Content factory, information warehouse, mental black box, workflow machine, automated daily reports.

How to draw: one large strange machine or device, small amount of input, one output. Must be memorable.

### Method Layering

Best for: Methodology frameworks, system hierarchies, capability stacks, layered content systems.

How to draw: boxes stacked in layers — no formal pyramids; Xiaohei moving bricks or building alongside.

### Map Route

Best for: From idea to launch, user paths, content handoff paths, learning paths.

How to draw: one curved route, few nodes, Xiaohei pulling a line or walking along it.

### Mini-Comic Panels

Best for: Failure to success, real processes, complaints, before/after usage changes.

How to draw: 2-4 small scenes, each expressing only one action.

## Original Metaphor Generation Method

Reinvent a metaphor from the current article every time — never copy old images.

### Three Steps

1. Replace an abstract concept with a physical action: getting stuck, leaking, becoming heavier, sorting, fermenting, precipitating, opening a door, folding, unpacking, flowing back.
2. Replace a system structure with a low-tech object: broken machine, cardboard box, drawer, water pipe, mailbox, strange dial, scale, well, ladder, strange workbench.
3. Make Xiaohei perform the action: not standing aside, but stuck inside a machine, pulling the wrong line, guarding a door, carrying, repairing, weighing, holding a ladder up, recording, stuffing something into a strange device.

### Available Object Pool

- Cardboard boxes, drawers, old machines, funnels, scales, mailboxes, doors, wells, ladders, water pipes, thread spools, gates, turntables, black boxes, hole punchers, pasta presses, clotheslines, strange workbenches.
- Only pick 1-2 per image — don't overcrowd.

### Xiaohei Action Pool

- Pull, carry, stuff, scoop, press, weigh, sew, cut, twist, guard, push, catch, dismantle, mark, recycle.
- Actions should serve the core meaning — don't be strange for the sake of being strange.

## Anti-Replication Rules

Don't default to opening or replicating `assets/examples/`. These images are for style calibration only — line density, white space, color restraint, Xiaohei's demeanor.

Unless the user explicitly says "copy this one / replicate this composition / modify based on this example", don't directly reuse these known compositions:

- Conveyor belt with two breakpoints
- Xiaohei pulling a judgment lever inside a content machine
- Xiaohei as a funnel sorting traffic/trust/conversion
- Xiaohei cutting a material fish
- Xiaohei leading a handoff path
- Xiaohei pulling three layers of information sources
- Three Xiaoheis holding a megaphone / building a bridge / opening a door
- Xiaohei stamping a script toolbox
- Xiaohei holding a sign on a common-pitfalls path

For similar themes, always invent new metaphors. For example, "handoff path" doesn't have to be a route — it could be Xiaohei attaching a content tail to a doorknob. "One fish, many uses" doesn't have to be a fish — it could be Xiaohei pressing a paper ball into several different shapes.
